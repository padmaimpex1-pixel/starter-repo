# starter-repo

Starter repository for onboarding local Windows users.

## Tools

### Top-25-CDrive-Usage.py

Scans C drive and reports the top 25 largest files and folders.

```powershell
python Top-25-CDrive-Usage.py
```

Output:
- Console table with sorted results
- CSV report: `C:\Users\Public\top-25-c-drive-usage.csv`

## Quick start

```powershell
git clone https://github.com/padmaimpex1-pixel/starter-repo.git
```
